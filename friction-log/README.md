# Friction Log

Raw pain points discovered while building apps against the ZeroBias SDK/client stack. This is the most important folder in the repo — every entry here is a candidate for improving the developer experience.

## File Naming

`{date}-{slug}.md` — e.g., `2026-04-08-client-auth-flow-unclear.md`

## Template

```markdown
---
status: draft
author: clark
app: sme-mart
severity: high
---

# {Short title}

## What I was trying to do

## What happened

## What I expected

## Workaround (if any)

## Suggested fix
```

## Severity Levels

- **critical** — Blocked development, no workaround
- **high** — Significant friction, workaround exists but painful
- **medium** — Confusing or undocumented, figured it out eventually
- **low** — Minor annoyance, nice-to-have improvement
